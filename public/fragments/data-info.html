• Data or Information
  • These works aim to highlight the significance of contextualizing content. Does the user know how--what--why this selection was made? Does the user have the ability to reconfigure that content? Is the user aware of the constraints which defined this set of information. Navigating the contemporary landscape requires manueivering through complex architectures of information. Does data exist anymore? Is data just information that offers users easy abiliy to reorganize and reconfigure information?

March 2013 Federer over Desktop, 2013. 42"x 32".
